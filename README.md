# 麻績勝広 ポートフォリオサイト

洋画家・麻績勝広（おみ かつひろ）さんの公式ポートフォリオサイト（MVP・日本語版）。
コンセプトは **「石と水と、その静けさ。」**。作品を主役にした、静謐な美術館的デザイン。

---

## 1. ファイル構成

```
omi-katsuhiro-portfolio/
├── index.html         トップ
├── works.html         作品（海／石・水石／泡）
├── about.html         作家について（略歴・年譜・受賞歴・展示）
├── exhibitions.html   展示・お知らせ
├── contact.html       お問い合わせ
├── assets/
│   ├── css/style.css  デザイン（色・文字・余白を一括管理）
│   ├── js/main.js     フェードイン等の軽い動き
│   └── img/           作品画像（＋出典メモ credits.md）
└── README.md          このファイル
```

特別なソフトは不要。**HTMLファイルをダブルクリック**すればブラウザで開けます。

---

## 2. 公開（ホスティング）の方法

静的サイトなので、フォルダごとアップロードするだけで公開できます。おすすめは無料で簡単な以下：

- **Netlify Drop** … <https://app.netlify.com/drop> にフォルダをドラッグ＆ドロップ
- **Vercel** / **GitHub Pages** / **さくらのレンタルサーバ** 等でもOK

独自ドメイン（例：`katsuhiro-omi.com` のような作家名のドメイン）を取ると、検索したときの信頼感が上がります。

## 3. よくある更新のやり方（非エンジニア向け）

ファイルは「メモ帳」や「テキストエディット」で開いて編集できます。
**編集前に必ずフォルダごとコピーしてバックアップ**を取ってください。

### A. 展示・お知らせを追加する
`exhibitions.html` を開き、`<li class="exh-item reveal">` 〜 `</li>` のかたまりを
ひとつコピーして、日付・展示名・会場を書き換えます。
トップページにも載せたいときは `index.html` の同じ部分も直します。

### B. 作品画像を差し替える / 追加する
1. 新しい画像を `assets/img/` に入れる（できれば長辺1600px以上の高解像度）。
2. `works.html` の `<img src="assets/img/ここ.jpg" ...>` のファイル名を書き換える。
3. `alt="..."`（画像の説明）と、その下の作品名・制作年・サイズも正しく直す。
   - **alt は必ず書く**：目の不自由な方や検索エンジンが作品を理解するために必要です。

### C. 「準備中（プレースホルダ）」の枠を本物の画像にする
`works.html` の「海」シリーズにある
`<div class="artwork__frame artwork__frame--placeholder">...</div>` を、
ほかの作品と同じ `<div class="artwork__frame"><img ...></div>` の形に置き換えます。

### D. 色や文字の雰囲気を変えたい
`assets/css/style.css` のいちばん上 `:root { ... }` にある色コード（`#F4F2ED` など）を
変えると、サイト全体の色が一括で変わります。

## 4. 公開前に直したい「仮」の部分

| 場所 | 内容 |
|---|---|
| `contact.html` | メールアドレスが仮（`example@example.com`）。正式な連絡先かフォームに差し替え |
| `works.html` | 一部作品のタイトル・制作年・サイズが「確認中」。本人確認のうえ記入 |
| `assets/img/` | 画像は暫定（Webから取得した低〜中解像度）。本人提供の高解像度に差し替え推奨 |
| 各ページ末尾 | `最終更新：2026.06` を更新時に直す |
| 各HTMLの `<head>` 内 JSON-LD | 構造化データ内の `https://example.com` を**独自ドメイン**に置き換える（SEO向上） |

## 4-2. 搭載済みの機能（メモ）

- **作品の拡大表示（ライトボックス）**：作品やチラシをクリックすると暗転して拡大。
  もう一度画像をクリックで**ズーム**、`← →`キーで前後送り、`Esc`または背景クリックで閉じる。
  追加ライブラリ不要（`assets/js/main.js` 内で完結）。新しい作品を `.artwork__frame` の形で追加すれば自動で対象になる。
- **カスタムカーソル（泡のような白い光）**：マウス端末で、中心からふわっと透けてフェードする半透明の白い球が
  少し遅れて追従（作品の「泡」のイメージ）。リンクや作品の上でやわらかく大きくなり、作品上では「View」の文字が出る。
  **スマホ/タブレットでは自動で無効**（指操作にカーソルはないため）。「動きを減らす」設定の人にも出さない。
  目立ちすぎ・重さを感じたら `assets/js/main.js` の「カスタムカーソル」ブロックと、
  `assets/css/style.css` の「カスタムカーソル」ブロックを削除すれば元に戻せる。
- **構造化データ（JSON-LD）**：各ページの `<head>` に作家情報・作品・展覧会を機械可読で記述済み。
  検索エンジンが「麻績勝広という人物・作品」を正しく認識しやすくなる。作品を追加したら `works.html` の
  `VisualArtwork` にも同じ要領で1件足すと良い（任意）。
